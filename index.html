Auth & Notification System

✉
Email Verification
Secure token-based email verification with 15-minute expiry, hashed storage, and beautiful HTML email templates.
  • Secure token generation (crypto.randomBytes)
  • 15-minute expiration window
  • Hashed token storage (bcrypt)
  • Resend with rate limiting
  • Prevent duplicate registration
📱
Phone OTP (30 sec TOTP)
Time-based OTP via TOTP algorithm — never stored in plaintext. 30-second validity with 1 retry, brute-force protected.
  • TOTP algorithm (RFC 6238)
  • 30-second validity window
  • Never stored in plaintext
  • 1 retry attempt allowed
  • Rate limiting per phone
💳
Payment & GST Invoice
Auto-generate PDF tax invoices with GSTIN, CGST/SGST/IGST on successful payment, emailed instantly.
  • Unique invoice number gen
  • PDF with CGST/SGST/IGST
  • Auto email on payment
  • Downloadable at any time
  • GST-compliant format

// SECURITY.05

Security Architecture

🔒

HTTPS Only

All traffic encrypted via TLS 1.3. HTTP redirects to HTTPS.

🔐

Hashed Tokens

Email tokens hashed with SHA-256. Passwords with bcrypt rounds=12.

⏱

Rate Limiting

5 OTP requests/hour per IP. Email resend max 3/hour.

🛡

Brute Force Guard

OTP locked after 2 wrong attempts. Account locked after 5 failed logins.

📋

Audit Logging

All auth attempts, OTP requests, payments logged with timestamps & IPs.

🌍

Phone Validation

E.164 format enforced. Country code required. Duplicate check active.

🔑

Env Variables

All secrets in .env. Never hardcoded. Rotated via vault.

📦

No Plaintext OTP

TOTP secret encrypted at rest. OTP computed on-demand, never stored.
